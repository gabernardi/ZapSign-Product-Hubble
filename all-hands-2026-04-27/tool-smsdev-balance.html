Ferramenta 01 · SMS Dev
Verificador de saldo SMS Dev
Consulta o saldo de SMS na conta SMS Dev e alerta quando está abaixo do limite configurado. Sua chave de API fica salva apenas no localStorage deste navegador — nada sai daqui.
Consulta
GET api.smsdev.com.br/v1/balance
Sobre a ferramenta
Quando usar
Antes de campanhas com volume alto de SMS, ou de tempos em tempos para garantir que a conta não vai zerar.
Limite padrão
Alerta de saldo baixo a partir de 500 SMS. Ajustável no campo acima.
Onde gerar a chave
No painel do SMS Dev, em Configurações → API.
Privacidade
A chave fica salva só no seu navegador (localStorage). A consulta é feita direto do seu browser para a API do SMS Dev.
Alternativa via terminal
caso o navegador bloqueie a chamada (CORS)
Se a consulta acima falhar com erro de rede, é provável que a API esteja bloqueando chamadas do navegador (política de CORS). Nesse caso, rode um destes comandos no terminal:
curl
# troque SUA_CHAVE pela sua chave SMS Dev curl -s "https://api.smsdev.com.br/v1/balance?key=SUA_CHAVE"
Python (script da skill)
# exporta a chave e roda o script empacotado na skill export SMSDEV_API_KEY="SUA_CHAVE" python <SKILL_DIR>/scripts/check_balance.py --threshold 500
